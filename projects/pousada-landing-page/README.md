# Pousada Serra Verde — Landing Page

Landing page moderna para uma pousada conceitual localizada em Lumiar, Nova Friburgo (RJ).  
Este projeto foi desenvolvido para portfólio com o objetivo de demonstrar habilidades em desenvolvimento front-end e criação de interfaces para o setor de turismo e hospedagem.

![Preview do site](img/preview.jpg)

## Tecnologias

- HTML
- CSS
- JavaScript

## Sobre o projeto

A página apresenta uma estrutura típica de landing page para pousadas e hospedagens, incluindo:

- Seção hero de apresentação
- Acomodações
- Experiências oferecidas
- Sobre a pousada
- Depoimentos de hóspedes
- Formulário de contato / reserva
- Integração com WhatsApp
